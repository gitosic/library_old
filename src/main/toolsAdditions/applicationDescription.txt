# Суть приложения "library"
Есть библиотека, в которой хранятся книги. Можно добавлять книги, введя:
1. Название книги (Name book)
2. Международный номер (ISBN)
3. Автор (Author)
Каждой книги присваивается читатель (Book reader), если он есть.
Можно удалить книгу из базы. Данные по каждой книги можно редактировать.

## Роли
1. Неавторизованный пользователь - Видит стартовую страницу.
2. Пользователь ("user") и администратор ("admin") - Видит все данные по книгам и у кого из читателей находится.
3. Администратор ("admin"). Может добавлять/удалять книги, вносить изменения по каждой книге.

### Проделанная работа
1. Развернул локально базу данных postgres.
2. С нуля написал приложение, которое работает с базой данных.
3. Разграничил в приложении возможности пользователей по ролям.
